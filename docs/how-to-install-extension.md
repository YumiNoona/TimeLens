## How to install the TimeLens browser extension### Chrome / Brave / Arc / Edge (Chromium)1. Download `TimeLens-extension-chrome.zip` from the [latest release](https://github.com/anomalyco/TimeLens/releases/latest)2. Extract the zip to a permanent folder (don't delete it after installing)3. Open your browser and go to:   - **Chrome/Brave/Arc**: `chrome://extensions`   - **Edge**: `edge://extensions`4. Enable **Developer mode** (toggle in the top-right corner)5. Click **Load unpacked** and select the extracted folder6. The extension icon appears in your toolbar ΓÇö click it to open the dashboard> Once approved, the extension will be available directly from the Chrome Web Store and Edge Add-ons store.### Firefox / Zen1. Download `TimeLens-extension-firefox.zip` from the [latest release](https://github.com/anomalyco/TimeLens/releases/latest)2. Extract the zip to a permanent folder3. Open Firefox and go to `about:debugging#/runtime/this-firefox`4. Click **Load Temporary Add-on**5. Select `manifest.json` inside the extracted folder6. The extension is active until you restart Firefox> For permanent installation, the extension must be signed by Mozilla. Until then, you can set `xpinstall.signatures.required = false` in `about:config` (Firefox Developer Edition / Nightly only), or wait for the [AMO listing](https://addons.mozilla.org).### How it worksOnce installed, the extension automatically sends browser tab information (URL, domain, title, audible state) to the local TimeLens API at `http://127.0.0.1:47821`. All data stays on your machine.
